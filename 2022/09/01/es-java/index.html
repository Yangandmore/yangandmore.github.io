    二、搜索语法入门

    2.1 query string searc

    无条件搜索

    • took: 花费多长时间
    • time_out：是否超时
    • _shards：
      • total：总共分片数
      • successful：成功分片数
      • skipped：跳过分片数
      • failed：失败分片数
    • total：查询总数
    • max_score：就是docuemnt对于一个search的相关度的匹配分数，越相关，就越匹配，分数就越高。
